# 🌍 React on the Go

A modern React application built with **Vite** that allows users to explore country data. This project was developed to practice component architecture, API integration, and deployment workflows.

## 🚀 Live Demo
Check out the live site here: [glistening-treacle-818182.netlify.app]

---

## ✨ Features
* **Dynamic Search:** Find countries instantly as you type.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
* **Component-Based Architecture:** Built using functional components and React hooks.
---

## 🛠️ Tech Stack
* **Framework:** React 19 (Vite)
* **Styling:** CSS3 / Modules
* **Deployment:** GitHub Pages
* **Version Control:** Git
